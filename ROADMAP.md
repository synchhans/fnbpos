# Roadmap fnbpos

Roadmap ini fokus MVP yang cepat dipakai (impact tinggi) tanpa kebablasan scope.

## Phase 0 - Foundation (MVP-Ready)
- [ ] Auth + Role-based access (OWNER, CASHIER, KITCHEN)
- [ ] Master menu: categories, items, modifier groups/options
- [ ] Orders: create -> send to kitchen -> status update -> complete/void
- [ ] Shifts: open/close + cash in/out + selisih
- [ ] Payments (basic): cash/transfer/qris-manual status
- [ ] Reports basic: sales daily, top items, shift summary
- [ ] Audit/event log untuk order status

## Phase 1 - Operasional Lebih Nyaman
- [ ] Table management (opsional: dine-in)
- [ ] Discount rules + approval (opsional)
- [ ] Reprint receipt + void rules
- [ ] Multi-station kitchen (minuman/makanan)
- [ ] Export CSV/Excel

## Phase 2 - Scale & Integrations
- [ ] Inventory bahan baku (COGS, stok)
- [ ] Multi-outlet
- [ ] Payment gateway + reconciliation
- [ ] Integrasi printer ESC/POS melalui desktop agent (stabil)

## Non-goals (untuk menjaga scope)
- Integrasi delivery aggregator di MVP
- Akuntansi lengkap (ledger, jurnal)
- Promo kompleks (bundling multilayer) di MVP
